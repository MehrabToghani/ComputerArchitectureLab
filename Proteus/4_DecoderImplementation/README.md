## Question

F3. The simplified Boolean functions for the circuit are:
1. F1 = xz + x!y!z!
2. F2 = x!y + xy!z!
3. F3 = xy + x!y!z

Implement and test the combinational circuit, using a 74155 decoder IC and external
NAND gates.
